## Description

This is an online pizza shop developed on React.

## Stack

```
- ReactJS + hooks
- React Router
- Redux
- Redux thunk
- json-server
- Axios
- classnames
```

## Installation

Step 1 — clone this repository

```
$ git clone https://github.com/rafagabidulin/react-pizza.git
```

Step 2 — install the dependencies

```
$ npm install
```

Step 3 — run the project

```
$ npm start
```
